# Weather App

A Flask-based weather dashboard using OpenWeather API.

## Features
- Current weather
- 5-day forecast
- Air quality index
- UV index
- Hourly temperature chart
- Dark mode

## Tech stack
- Python (Flask)
- JavaScript
- Chart.js
- TailwindCSS
- OpenWeather API
